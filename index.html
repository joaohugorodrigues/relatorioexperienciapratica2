Relatório de Análise do Mercado de TI

Introdução

Este relatório tem como objetivo analisar o mercado de trabalho na área de Desenvolvimento Web, com foco em posições júnior e pleno. A pesquisa foi realizada em plataformas como LinkedIn e Glassdoor, e os dados foram coletados a partir de cinco vagas.

1. Habilidades Técnicas (Hard Skills)

As habilidades técnicas são o alicerce para qualquer vaga de desenvolvimento. A análise das cinco vagas revelou as seguintes competências mais exigidas:

  • JavaScript: Presente em 100% das vagas. É a linguagem mais fundamental.
  • HTML e CSS: Mencionado em 40% das vagas, mas é um conhecimento base para qualquer desenvolvedor front-end.
  • APIs (REST): Presente em 60% das vagas. A capacidade de desenvolver e consumir APIs é crucial.
  • Git/GitHub: Presente em 60% das vagas. A familiaridade com versionamento de código é um requisito padrão.
  • Python: Mencionada em 40% das vagas (em contexto de Django e back-end).
  • SQL: Presente em 40% das vagas. Conhecimento em bancos de dados é essencial para Full Stack.
  • Outras Tecnologias: N8N/Make, NextJS, Flutter, Kotlin, Java, Power Automate.

2. Habilidades Comportamentais (Soft Skills)

As soft skills são cruciais para o trabalho em equipe e o crescimento profissional. As vagas indicaram as seguintes habilidades como as mais importantes:

  • Comunicação e Colaboração: Presente em 80% das vagas (mencionada explicitamente ou implícita em "trabalho em equipe").
  • Vontade de Aprender: Mencionada em 40% das vagas ("curiosidade", "aprender rápido").
  • Proatividade: Presente em 40% das vagas ("mão na massa", "visão sistêmica").

4. Tendências do Mercado

  • Trabalho Remoto: Quatro das cinco vagas oferecem a modalidade de trabalho 100% remoto, o que é uma forte tendência no mercado de TI.
  • Automação e Integração: Ferramentas como N8N, Make e Power Automate estão em alta, mostrando a importância de desenvolvedores com habilidades em conectar sistemas.
  • Sistemas Legados: A modernização de sistemas antigos (legados) é uma necessidade, com o conhecimento de GCP e outras plataformas de nuvem sendo um diferencial.
  • Tecnologias Específicas: Conhecimento em frameworks como NextJS é cada vez mais valorizado.

5. Plano de Ação

Com base na análise, o plano de ação a seguir tem como objetivo desenvolver 3 habilidades-chave para se destacar no mercado de TI nos próximos 6 meses.

  • Habilidade Técnica 1: JavaScript Avançado e APIs
    • Meta: Dominar conceitos como Promises, `async/await`, e o consumo de APIs REST de forma eficiente.
    • Ação: Construir um projeto pessoal que interaja com uma API pública (ex: API de previsão do tempo ou de filmes).
  • Habilidade Técnica 2: React e NextJS
    • Meta: Aprender a criar componentes em React e entender os conceitos básicos do framework NextJS, que é um diferencial.
    • Ação: Criar um portfólio pessoal ou um blog utilizando NextJS.
  • Habilidade Comportamental 1: Comunicação e Colaboração
    • Meta: Melhorar a comunicação e o trabalho em equipe.
    • Ação: Contribuir com código e participar de discussões no projeto de código aberto Conecta+ no GitHub, ou participar de hackathons e eventos da comunidade.
